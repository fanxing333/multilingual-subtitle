# multilingual-subtitle
基于 **gpt-3.5-turbo** 将 YouTube 自动生成的英文字幕、Whisper 自动识别的英文字幕转换为中文字幕或中英双语字幕脚本。 为什么不选择 YouTube 的中文字幕？传统的流式音频识别会有一些突兀的事实错误，且中文实时翻译的效果更差。希望 gpt 可以通过学习上下文来避免此类错误。
